Dub.co Conversion Tracking Snippets

Copy and paste these snippets for your clients to implement conversion tracking.

Important: Replace yourdomain.com with your actual domain name in all snippets below.

1. General Page Tracking (Add to ALL pages)

Add this snippet to the <head> section of all pages where you want to track clicks:

<!-- Dub.co Analytics Tracking -->
<script>
(function(d,s,id,domain){
    if(d.getElementById(id)) return;
    var js=d.createElement(s),fjs=d.getElementsByTagName(s)[0];
    js.id=id;js.async=true;
    js.src='https://'+domain+'/js/dub-tracking.js';
    fjs.parentNode.insertBefore(js,fjs);
})(document,'script','dub-tracking-js','yourdomain.com');
</script>

2. Basic Conversion Tracking (Add to thank you pages)

Add this snippet to conversion/thank you pages:

<!-- Dub.co Conversion Tracking -->
<script>
window.DubConversionConfig = {
    eventName: 'Sign up'  // Change this for each conversion type
};
</script>
<script>
(function(d,s,id,domain){
    if(d.getElementById(id)) return;
    var js=d.createElement(s),fjs=d.getElementsByTagName(s)[0];

</script>

    <!-- General page tracking -->

</body>
</html>

Testing Instructions for Clients:

  1. Add the general tracking snippet to a test page
  2. Visit the page with ?dub_id=test123 in the URL
  3. Go to a page with the conversion snippet
  4. Open browser developer tools and check the Console tab for success messages
  5. Look for messages like "Dub Conversion: Tracked successfully"
